# redux 练习

### 初始化

``````javascript

 npm install

``````

### 实例

``````javascript

 npm start

 http://localhost:3000 是一个todo 实例

``````
### 打包

``````javascript
  gulp clean
  gulp  build

  localhost:3000 todo实例
  locaohost:3000/login  异步获取数据的例子
``````

  //开发环境
